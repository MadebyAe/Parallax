## Setup

---
**Requirements**

`node`

`babel`

`npm`

**Installation**

Instalamos todas las dependencias tanto de desarrollo como para producción.

`npm install` 

NPM en modo desarrollo

`npm run watch`

NPM en modo producción

`npm run build`
